## VUE-MOBILE-BONE

> 本工程基于 vue + vantUI + vuex 搭建一套前端移动端的基础开发框架

## **环境准备**

适配 vue-cli@3.x

- git
- vue-cli
- node@^8.12.0(development)

## **快速构建**

### **开发环境启动**

```bash
# 获取repo
git clone https://github.com/lifanlingjue/vue-mobile-bone.git
# 安装依赖
npm install
# 运行前端服务
npm run serve
```

### **生产环境部署**

```bash
# 工程打包
npm run build
```
